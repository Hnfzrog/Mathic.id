# Scope
Layout and section definitions for the landing page.

# Goals
- Provide a smooth scrolling experience with clear sections.

# Non Goals
- Multi-page routing.

| 3 | **Tentang**      | Service description, SVG avatar illustration (privacy), brand values |
| 4 | **Program**      | 3 cards: SD (1-6), SMP (7-9), SMA (10-12) with topics |

| 7 | **Footer**       | Logo, small tagline, WA icon, copyright © 2025 mathic.id |
