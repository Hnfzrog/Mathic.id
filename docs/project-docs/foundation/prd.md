# Description
mathic.id is a portfolio and landing page website for a private math tutoring service, catering to students from elementary to high school with flexible scheduling.

# Important
- Zero backend, purely static website.
- Registration uses a WhatsApp deep link.

# Scope
Development of a single-page static website (HTML/CSS/Vanilla JS) acting as a landing page and portfolio for mathic.id.

# Goals
- Provide a professional, trustworthy, and modern web presence.
- Allow prospective students/parents to easily contact mathic.id via WhatsApp.
- Showcase the tutoring programs (SD, SMP, SMA) and their benefits.

# Non Goals
- No backend systems or databases.
- No direct payment gateway integration.
- No real tutor photos (use SVG avatars for privacy).
- No pricing displayed on the website.

# Personas
- **Parents** (age 30-50): Searching for reliable and capable tutors for their children.
- **Students** (age 13-18): Looking for independent learning assistance for Middle School or High School math.

# Features
1. **Landing page:** Single-page design with multiple sections.
2. **Registration Form:** A form that captures Name, Grade, and Topic, which then redirects to WhatsApp.
3. **CTA (Call to Action):** "Hubungi kami" to ask for pricing and details via WhatsApp.
4. **Privacy:** Tutor photos are replaced with illustrations/avatars. Testimonials are replaced with a "Keunggulan Kami" (Our Strengths) section.
